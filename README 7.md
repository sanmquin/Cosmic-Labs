# Senior Full-Stack Engineer

**Location:** San Francisco (Onsite)  
**Type:** Full-time  
**Start Date:** ASAP

---

## About the Role

Cosmic Labs is hiring a senior full-stack engineer to build the user-facing experience and application layer of our hardware-aware desktop environment and AI copilot for embedded and network engineers.

This role is focused on the upper stack: desktop app, interfaces, APIs, and integration of AI agents into everyday engineering workflows.

---

## What You'll Do

- Build and maintain the Spades desktop application (Electron + TypeScript + React)
- Design and implement clean, responsive interfaces
- Integrate AI agents and MCP-style context flows into the product experience
- Develop backend services and APIs in Python to connect hardware, agents, and UI
- Implement real-time visualizations
- Collaborate closely with systems and embedded engineers
- Help define product behavior, UX patterns, and engineering best practices
