# Founding Engineer – Fullstack Engineer

**Location:** San Francisco (Onsite)  
**Type:** Full-time  
**Start Date:** ASAP

---

## What You'll Do

* Build the web and desktop interfaces for our hardware intelligence platform
* Design and implement real-time UIs for hardware visualization, terminal multiplexing, and device orchestration
* Develop APIs and backend services that bridge frontend experiences to low-level hardware systems
* Create intuitive developer tools and dashboards for embedded systems engineers
* Work on AI copilot interfaces and agent-driven workflows
* Own the full product experience from design to deployment
* Collaborate with systems engineers to surface complex hardware state in accessible ways

---

## What You Bring

**Strong experience with:**

* Modern frontend frameworks (React, Vue, Svelte)
* TypeScript and JavaScript ecosystem
* Backend development (Node.js, Python, Go)
* Real-time systems and WebSocket/streaming architectures
* Database design (PostgreSQL, Redis, time-series databases)
* API design and implementation (REST, GraphQL, gRPC)

**Familiarity with:**

* Desktop application development (Electron, Tauri)
* Terminal emulators and CLI tooling
* Low-level system integration from frontend/backend
* Data visualization for real-time systems
* AI/LLM integration and agent frameworks

## What We're Looking For

We're looking for a fullstack engineer who can build beautiful, responsive interfaces for systems that touch real hardware — where milliseconds matter and reliability is non-negotiable.

**You should have:**

* Strong product sense and attention to developer experience
* Experience building real-time, data-intensive applications
* Ability to work across the full stack and make pragmatic tradeoffs
* Comfort diving into unfamiliar domains (hardware, embedded, networking)
* Excitement about building tools that hardware engineers actually want to use

**Requirements:**

* Bachelor's or Master's in Computer Science or equivalent experience
* 3+ years of fullstack development experience
* Willingness to work startup hours, in-person (weekends included) at our San Francisco office
* Work authorization in the United States
